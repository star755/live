## 提示 ##
1、由于直播管理后台尚未上线，修改appkey会因为没有主播权限等问题导致不能正常直播  
2、未正常关闭的直播(被Kill、Crash、关闭直播的时候网络差等原因)，会一直显示为正在直播中，导致首页一直能看到这个直播室，进去之后看到是黑屏

## 简介 ##
本demo演示了通过环信及ucloud sdk实现视频直播及直播室聊天
